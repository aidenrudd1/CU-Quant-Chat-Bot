# CU-Quant-Chat-Bot
Chat Bot Being designed for CU Quants

The Goal of our chat bot is to allow a user to familiarize themselves with our already existing web page. Allowing them to understand the club, what we do, and the members of our team. 

The Github Repo has a linked Page that takes you to a demo web page, not our actual club website. This is because the chat bot while operational, still has bugs and performance issues that need testing an dfine tuning before live deployment to production. I will link both of these webpages here. 

Demo Webpage: https://aidenrudd1.github.io/CU-Quant-Chat-Bot/
CU Quants Main Webpage: https://www.cuquants.com/home

Also in this repository there is a Midterm write up. This documents presents our thoughts, ideas, chat bot structure, and provides example transcripts for a curious mind to read through. However, we hope and suggest you open up the demo page and try it yourself!
